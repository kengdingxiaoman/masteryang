# idea 命令积累
只针对 mac 环境

### 导入类
option + 回车

### 删除一行
command + delete

### 回到上一个光标处
command + option + 左右键

### 切换窗口
command + `

### 跳转到实现
option + shift + b

### command + N
command + N (Generate )快捷键就是实现各种方法

先点击 command + N，然后选择具体要实现的

可以实现
- getter and setter
- 构造方法
- 接口中方法定义的实现
- 重写父类中的方法
- 重写 toString() 方法

## 优化

### 优化import
ctrl + option + o

可以删除未引用的类

### 优化代码格式

option + command + l

## 查找

### 查找类
command + o

### 查找文件
shift + command + O

### 全面搜索
shift + command + f

在整个项目中查询，很好用

### 跳转到行
command + l

### 查看类中的所有方法
command + F12

如果你 disable了 fn 键

则使用 fn + command + F12
